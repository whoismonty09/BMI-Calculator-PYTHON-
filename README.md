# ⚖️ BMI Calculator (Python)

## 📌 Project Overview
This project is a simple Python based BMI Calculator that computes Body Mass Index using user weight and height and displays the health category.

---

## 🚀 Features
- Calculates BMI value
- Shows health status such as underweight normal overweight or obese
- Simple command line interface
- Fast and easy to use

---

## 🛠 Technologies Used
- Python

## 📚 Concepts Applied
- User input handling
- Mathematical calculations
- Conditional statements
